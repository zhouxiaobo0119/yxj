方法详解 #

getAppBaseInfo #

获取用户基本信息

调用 #

var res = window.__yxj_app.getAppBaseInfo()

返回值 res：promise对象 #

window.__yxj_app.getAppBaseInfo().then(userInfo => {
  // userInfo - 用户信息
  console.log('用户信息-------',userInfo)
})

// 或者

var userInfo = await window.__yxj_app.getAppBaseInfo()
// userInfo - 用户信息
console.log('用户信息-------',userInfo)

appHandler #

方法对象

调用 #

var res = window.__yxj_app.appHandler.方法()

方法 #

getAppBaseInfo, // 获取app用户基本信息,同上,window.__yxj_app.getAppBaseInfo

setTitle, // 设置标题 【点击跳转】

setTitleColor, // 设置标题背景颜色 ios设置透明度无效 【点击跳转】

setNavTitleColor, // 设置标题颜色 ios设置透明度只支持1或非1 【点击跳转】

toLogin, // 跳转登录 【点击跳转】

setShare, // 设置分享信息 在右上角显示 【点击跳转】

invokeShare, // 调起分享功能 （默认分享逻辑 不传参数 取setShare方法配置的参数 【点击跳转】

invokeShareForOwn, // 调起分享功能 针对单独分享逻辑（传入分享参数） 【点击跳转】

setBarRightVisibility, // 设置标题右侧文字是否隐藏 【点击跳转】

setBarLeftVisibility, // 设置标题左侧文字是否隐藏 【点击跳转】

modifyRightButtonTitle, // 修改右上角按钮文字, 及android 的回调 【点击跳转】

setInterceptBarEvent, // 方法_设置是否拦截TitleBar上面的点击事件 【点击跳转】

jumpToVc, // 跳转到app的任意页面并触发回调 【点击跳转】

modifySharePlatforms, // 设置分享平台 【点击跳转】

closeView, // 关闭webView 【点击跳转】

AuthStep1Activity, // 跳转到去认证，暂不入库，如有疑问，咨询开发人员

hiddenTitleBar, // 隐藏标题栏 【点击跳转】

setTitle #

设置app标题

调用 #

window.__yxj_app.appHandler.setTitle('标题')

入参 #

设置的标题（string）

返回值 #

无

setTitleColor #

设置标题背景颜色 ios设置透明度无效

调用 #

window.__yxj_app.appHandler.setTitleColor({
  red: 255,
  green: 0,
  blue: 0,
})

入参 #

rgb对象

{
  red: 255,
  green: 0,
  blue: 0,
}

返回值 #

无

setNavTitleColor #

设置标题颜色 ios设置透明度只支持1或非1

调用 #

window.__yxj_app.appHandler.setNavTitleColor({
  red: 255,
  green: 0,
  blue: 0,
  alpha:0
})

入参 #

rgb对象 red: 红 0-255 green: 绿 0-255 blue: 蓝 0-255 alpha: 透明度 0 - 1

{
  red: 255,
  green: 0,
  blue: 0,
}

返回值 #

无

toLogin #

跳转登录，调用此方法app会默认退出登录，清除app登录态

调用 #

window.__yxj_app.appHandler.toLogin()

入参 #

无

返回值 #

无

setShare #

设置分享信息 在右上角显示

调用 #

window.__yxj_app.appHandler.setShare({
  title: '',
  desc: '',
  link: '',
  imgUrl:""
})

入参 #

object对象

obj:{
  title: '', // 分享标题
  desc: '', // 分享描述
  link: '',// 分享链接地址
  imgUrl:"" // 分享图片地址
}

返回值 #

无

invokeShare #

调起分享功能 （默认分享逻辑 不传参数 取setShare方法配置的参数

调用 #

window.__yxj_app.appHandler.invokeShare()

入参 #

无

返回值 #

无

invokeShareForOwn #

调起分享功能 针对单独分享逻辑（传入分享参数）

调用 #

window.__yxj_app.appHandler.invokeShareForOwn({
  title: '', // 分享标题
  desc: '', // 分享描述
  link: '',// 分享链接地址
  imgUrl:"", // 分享图片地址
  platform:'', //  分享平台
})

入参 #

object对象，同setShare入参

obj:{
  title: '', // 分享标题
  desc: '', // 分享描述
  link: '',// 分享链接地址
  imgUrl:"", // 分享图片地址
  platform:'', //  分享平台
}

返回值 #

无

clazzName_ios：ios类名 string clazzName_adr：安卓类名 string obj：安卓参数 object iosObj：ios参数 object otherParams: 其他参数 object

返回值 #

无
